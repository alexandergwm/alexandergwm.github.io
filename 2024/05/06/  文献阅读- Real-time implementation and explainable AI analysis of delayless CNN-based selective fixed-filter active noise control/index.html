文献阅读: Real-time implementation and explainable AI analysis of delayless CNN-based selective fixed-filter active noise control

发表于2024-05-06|更新于2024-07-03|文献阅读
|阅读量:

摘要

选择性固定滤波器有源噪声控制（SFANC）方法可以针对不同类型的噪声选择合适的预训练控制滤波器。借助卷积神经网络（CNN）的学习能力，基于CNN的SFANC方法可以自动从噪声数据中学习其参数。结合实践经验，本文将ANC抽象为马尔可夫过程，并提供详细的理论分析来验证基于CNN的SFANC方法的合理性。为了验证其有效性，我们在多通道 ANC 窗口中实现该方法，其中协处理器中运行的 CNN 与实时控制器协作，实现无延迟噪声控制。此外，使用可解释的人工智能技术来分析基于 CNN 的 SFANC 方法的基本原理，增强其在声学应用中的可解释性。数值模拟和实时实验表明，基于CNN的SFANC方法不仅对宽带和现实世界噪声实现了令人满意的降噪性能，而且具有出色的可传递性。

1. 介绍

​ 城市化和工业化导致环境噪声污染大幅上升。传统的被动噪声控制方法通过利用材料和物理屏障来减少噪声传播和吸收来解决这个问题[1]。然而，在降低低频噪声时，无源方法要么无效，要么成本高昂且体积庞大[2-6]。与无源噪声控制相比，有源噪声控制（ANC）技术可产生与不需要的噪声幅值相等但相位相反的抗噪声，可以在消除低频噪声方面提供优越的降噪性能[7-11] 。由于其紧凑的尺寸和方便的部署，ANC系统越来越多地应用于各种噪声敏感的商业产品，如窗户[12-15]、耳机[16-18]、头枕[19]和车辆[20,21] 。

​ 滤波参考最小均方 (FxLMS) 算法通常用于传统 ANC 系统，如图 1 所示。FxLMS 算法自适应调整控制滤波器系数以最小化误差信号 [22-24]。然而，自适应算法存在一些固有的局限性，包括收敛速度慢、跟踪能力差、潜在发散风险高等[25-28]。相比之下，采用预先训练的控制滤波器进行噪声消除的固定滤波器 ANC 方法具有更快的响应速度和更高的鲁棒性[29]。然而，固定滤波器 ANC 算法针对特定噪声类型进行了优化，导致在控制其他类型噪声时表现平平 [30,31]。为了解决这个问题，人们提出了一种选择性固定滤波器ANC（SFANC）方法，该方法可以针对不同的噪声选择合适的预训练控制滤波器[32]。然而，该方法的一些关键参数只能通过反复试验来确定。为了克服这一限制，深度学习技术，特别是卷积神经网络（CNN），已成为提高 ANC 性能的可行方法[33-36]。然而，现有的大多数基于深度学习的 ANC 方法都采用神经网络来取代 ANC 系统中的控制滤波器。这些神经网络的高计算复杂性超出了实时处理器的能力，导致处理延迟。巨大的处理延迟可能会违反 ANC 系统的因果关系要求，并导致噪声控制效果不佳。

​ 为了在实际场景中探索更有效的基于深度学习的 ANC 方法，一些研究提出了基于 CNN 的 SFANC 方法 [37-39]，其中利用 CNN 为传入噪声选择预训练的控制滤波器。凭借 CNN 的学习能力，所有参数都是从噪声数据中自动学习的，无需手动操作 [40]。尽管仿真表明基于CNN的SFANC方法可以有效地降低不同的噪声，但仍然缺乏实际的实现和分析。此外，在之前的模拟中，假设评估期间的主要路径和辅助路径与训练期间使用的路径相同。然而，实际应用中的声学路径可能与训练中使用的声学路径不同[41,42]。因此，仍然需要在实践中评估基于CNN的SFANC方法的降噪性能和可迁移性。

​ 在本文中，我们在宿舍窗口中实现了基于CNN的SFANC方法进行实时噪声控制，该方法可以通过协处理器和实时控制器之间的高效协调来实现无延迟降噪。在理论分析方面，本文将ANC抽象为马尔可夫过程，以验证基于CNN的SFANC方法的合理性。此外，我们使用可解释的人工智能技术，层类激活图（LayerCAM）[43]，来深入了解CNN模型的内部原理并解释其在SFANC中的有效性。仿真和实时实验表明，基于CNN的SFANC方法可以有效降低不同类型的噪声，并在实际场景中表现出良好的可迁移性。

​ 后续章节的组织如下：首先，第 2 节将 ANC 过程表述为马尔可夫模型，为基于 CNN 的 SFANC 方法提供理论基础。随后，第 3 节全面介绍了基于 CNN 的 SFANC 方法。此外，第 4 节介绍了一种可解释的人工智能技术。第 5 节和第 6 节分别提供了使用基于 CNN 的 SFANC 方法的数值模拟和实时实验。 最后，结论在第 7 节中给出。

3. 基于CNN的无延迟SFANC方法

​ 图 4展示了基于 CNN 的 SFANC 方法的框图，该方法由两个模块组成：协处理器中控制滤波器的选择和实时控制器中的噪声消除。在此方法中，轻量级 2D CNN 在协处理器中运行，为每个噪声帧选择最佳的预训练控制滤波器。随后，选定的控制滤波器被传送到以采样率并行运行的实时控制器。协处理器和实时控制器之间的有效协调可以在基于 CNN 的 SFANC 方法中实现无延迟噪声控制。

3.1 预训练控制滤波器

​ 获得预训练的控制滤波器是基于 CNN 的 SFANC 方法的第一步。在此阶段，我们利用目标 ANC 系统来消除宽带白噪声，其频带包含感兴趣的频率分量。由于计算复杂度较低，FxLMS 算法可用于导出针对这些宽带噪声的最佳控制滤波器。最后，将获得的预训练控制滤波器存储在控制滤波器数据库中。

3.4 实时噪音消除

​ 在整个噪声控制过程中，实时处理器逐个样本地进行噪声控制。同时，对于每个噪声帧，协处理器利用经过训练的 2D CNN 来选择最佳的预训练控制滤波器。如果选择的控制滤波器与当前使用的控制滤波器不同，则将用它来更新使用的控制滤波器，以进行后续的噪声控制。表2提供了基于CNN的SFANC方法中噪声控制过程的伪代码。

​ 总体而言，协处理器和实时控制器并行运行，这确保了实时噪声控制不会受到2D CNN处理延迟的影响。这种配置允许强大的批处理协处理器（例如笔记本电脑）运行 2D CNN，而实时控制器则优先考虑立即处理以保持低延迟。两个处理单元之间的高效协调可以实现无延迟的噪声控制，使该方法成为实际环境中可行的解决方案。

5.1 2D CNN的有效性

​ 本节首先介绍用于训练 2D CNN 的噪声数据集。然后，通过分析 2D CNN 的分类准确性和学习特征来评估 2D CNN 的有效性。另外，利用LayerCAM技术研究了2D CNN的滤波器选择原理。

5.1.1 网络的训练

​ 使用合成噪声数据集来训练 2D CNN，其中包含 80,000 个用于训练的噪声实例、2000 个用于验证的噪声实例以及其余 2000 个用于测试的噪声实例。合成噪声实例是通过具有随机选择的中心频率和带宽的各种带通滤波器过滤白噪声而生成的。每个噪声实例的持续时间为 1 秒。如(12)中所述，每个噪声实例的类别是7个预训练控制滤波器中实现最高降噪水平的控制滤波器的索引。

​ 在 2D CNN 的训练中，交叉熵损失被用作优化指标。此外，使用Adam算法[51]进行优化。训练epoch数设置为50。初始学习率设置为0.01，每10个epoch减少0.1倍。我们使用在 ImageNet 上训练的 ShuffleNet V2 [46] 模型的权重初始化了 2D CNN 的所有未修改层。

​ 在图7中，我们可以观察到不同类别的噪声特征在二维空间中很好地分离。尽管在少数情况下异常特征会被错误分类，但大多数噪声特征都表现出高度的聚类性。值得注意的是，不同类别之间存在显着差异，而同一类别内的差异相对较小。它证实了 2D CNN 学习的噪声特征具有扎实的判别能力。因此，特征可视化结果表明，SFANC方法中训练的2D CNN可以准确地对不同类型的噪声进行分类。符合2D CNN的高分类精度。

5.1.4使用 LayerCAM 进行可解释的 AI 分析

​ 前面的部分证实了 2D CNN 对于噪声分类的有效性。在本节中，我们采用 LayerCAM 技术 [43] 来研究 2D CNN 中噪声分类的基本原理。如第 4 节所述，LayerCAM 技术可以为不同的 2D CNN 层生成类激活图，从而突出显示输入梅尔频谱图图像中的类特定判别区域。音调噪声 (300 Hz) 和宽带噪声 (50 Hz-800 Hz) 的 LayerCAM 结果如图 8和9。2D CNN将两个噪声的类别分别预测为$B_3$和$B_0$。

根据图8（a）和图9（a）中的梅尔频谱图，音调噪声的频带比宽带噪声中观察到的频带窄得多。如图6所示，控制滤波器$B_3$和$B_0$的频率范围分别约为200-325Hz和200-700Hz。在 7 个预先训练的控制滤波器中，控制滤波器$B_3$ 和$B_0$ 的频谱与音调噪声和宽带噪声的频率范围最匹配。该结果表明基于 CNN 的 SFANC 方法中的滤波器选择主要基于输入噪声的频率范围。

​ 此外，通过比较图8和图9的(a)和(b)，我们注意到梅尔谱图图像中表现出高强度的区域对应于融合 LayerCAM 中的高激活值。这表明2D CNN的决策过程主要关注噪声的高强度频谱内容，即频带信息。因此，结果证实基于 CNN 的 SFANC 方法主要依靠噪声频带信息来选择控制滤波器，与[32]中提出的理论证据一致。此外，2D CNN 不同阶段的 LayerCAM 显示浅层（Conv0、Conv1、Stage2）主要关注局部特征，而更深层（Stage3、Stage4、Stage5）则关注全局特征。 2D CNN 可以利用局部和全局频率信息准确地对噪声进行分类。

6.基于CNN的SFANC的实时实现

​ 为了评估基于 CNN 的 SFANC 方法在实际场景中的性能，我们利用该方法实现了尺寸为 47 cm × 47 cm 的 4 通道 ANC 窗口。图 12 说明了该多通道 ANC 系统的示意图。在系统中，协处理器是一台笔记本电脑（配备 NVIDIA GeForce RTX 3060 笔记本电脑 GPU），它运行经过训练的 2D CNN 以选择预先训练的控制滤波器。同时，带有前置放大器、I/O单元和输出放大器的嵌入式PXI处理单元（NI PXIe-8135）充当降噪的实时控制器。在噪声控制过程中，笔记本电脑利用UDP协议[54]将所选控制滤波器的索引传输到PXI处理单元。在这种联合工作模式下，笔记本电脑以帧速率运行，在线噪声控制以16 kHz的采样率并行运行，这使得基于CNN的SFANC方法可以在ANC窗口中实现无延迟噪声控制。

6.1 实验设置

​ 4 通道 ANC 窗口的实验设置如图 13 所示。值得注意的是，没有用于实时噪声控制的误差麦克风。与自适应ANC算法不同，基于CNN的SFANC方法不需要误差信号来更新其控制滤波器，这使得其部署更加方便。

6.2 预训练控制滤波器

​ 最初，在具有 1 个参考麦克风、4 个次级源和 4 个误差传感器的 4 通道 ANC 窗口中获得 7 个预训练的控制滤波器。使用图5所示的7种不同频率范围的宽带噪声作为主要噪声。采用FxLMS算法来获得这些宽带噪声的最优控制滤波器。收敛后，得到7个预训练的控制滤波器，索引值范围为$B_0$到$B_6$。

​ 获得预训练的控制滤波器后，SFANC系统中可以进行无误差麦克风的实时噪声控制，从而便于实际部署。而且，通过避免误差信号的反馈机制，提高了响应时间和系统稳定性。在噪声控制期间，初始控制滤波器是索引为0的预训练控制滤波器。当2D CNN选择不同的预训练控制滤波器时，笔记本电脑将更新后的索引传输到PXI处理单元以加载相应的预训练控制滤波器。 此外，为了评估可转移性，在合成声学路径上训练的 2D CNN 模型（如第 5 节中详述）直接应用于真实声学路径，无需重新训练.

6.3 宽带噪声消除

​ 在第一个实验中，所实现的ANC系统用于消除7种宽带噪声，其性能如图15和图16所示。7种宽带噪声的频率范围如图5所示。实验结果表明，基于CNN的SFANC方法有效地衰减了宽带噪声的主要功率。具体来说，该技术在降低 325-450 Hz 和 450-575 Hz 噪声方面效果较差，可能是由于相应预训练控制滤波器在这两个频率范围上的性能有限。从降噪的1/3 倍频程来看，我们发现，噪声抑制值在大多数频率下为正值，但在特定频率下为负值，这可能是由实验室的声学模式引起的。

​ 表5总结了所选控制滤波器针对这些宽带噪声的降噪水平和指标。基于CNN的SFANC方法对325-450 Hz和450-575 Hz噪声实现了约12 dB的降噪水平。对于其他类型的宽带噪声，降噪水平超过15 dB。这些降噪效果的差异来源于不同的各自预训练控制过滤器的功效[55]。此外，所选的控制滤波器与这些宽带噪声的频率范围相匹配，证明 SFANC 方法中的 2D CNN 正确地对这些宽带噪声进行了分类。结果还表明，在合成声学路径上训练的 2D CNN 模型仍然可以准确地对真实声学路径上的噪声进行分类，而无需重新训练过程。因此，证明了SFANC中2D CNN模型的分类精度和可迁移性。

6.4 真实噪声消除

​ 在本实验中，使用基于CNN的SFANC方法的4通道ANC窗口来消除两种真实噪声：飞机噪声和压缩机噪声。值得注意的是，训练数据集中不存在真正的噪声。图 17 和表 6 分别显示了真实噪声的功率谱和降噪水平。

​

​ 图17结果表明，虽然两种噪声的功率谱表现出显着差异，但基于 CNN 的 SFANC 方法可以有效衰减两种噪声中的 200-700 Hz 频率分量。这是因为预训练控制滤波器的最大频率范围是 200-700 Hz。根据表6，基于CNN的SFANC算法对飞机噪声和压缩机噪声分别实现了12.29 dB和12.43 dB的降噪水平。因此，实验结果证明了基于 CNN 的 SFANC 方法在处理真实噪声方面的有效性。此外，如果声学路径发生显着变化，我们只需要针对新的声学环境获得相应的预训练控制滤波器。重要的是，不需要重新训练 2D CNN，这展示了基于 CNN 的 SFANC 方法在不同声学场景中的良好可迁移性。

7. 结论

​ 本论文主要强调了基于CNN的SFANC方法的实用性和可解释性。基于CNN的SFANC方法是一种数据驱动的方法，它在协处理器中采用2D CNN来针对不同类型的噪声选择合适的预训练控制滤波器。协处理器和实时控制器之间的高效协调可实现无延迟的噪声控制。在理论分析方面，本文将ANC抽象为马尔可夫过程，以验证基于CNN的SFANC方法的理论合理性。

​ 数值模拟表明所提出的 2D CNN 具有 98.55% 的高分类精度。使用可解释的人工智能技术 LayerCAM，我们发现 2D CNN 的分类主要取决于噪声的频带信息。 ANC窗口的实时实验表明，基于CNN的SFANC方法有效地衰减了宽带噪声和真实动态噪声。此外，在合成声学路径上训练但在真实环境中使用的 2D CNN 证明了所提出方法的良好可移植性。
