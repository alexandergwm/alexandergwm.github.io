解惑：为什么子空间算法中的导向矢量张成的信号子空间与噪声子空间正交

为什么子空间算法中的导向矢量张成的信号子空间与噪声子空间正交

MUSIC算法为代表的子空间算法算是信号处理和阵列信号处理下的一个常用的算法，其关键思想在于将接收到的信号分为两个正交子空间：信号子空间和噪声子空间。信号子空间包含的是从各个方向到达阵列的信号的信息，而噪声子空间主要包含噪声和干扰，当进行特征值分解（EVD）或者奇异值分解（SVD）时，这两个子空间由对应的特征向量定义。

那么为什么子空间算法中的导向矢量张成的信号子空间和噪声子空间正交？

我们从数学公式上来进行推导，先对每个符号的含义做一个定义。

当然，让我们将你的详细推导与我之前的建议结合起来，以构建一个全面而详细的解释和证明。

符号与定义

  • PPP : 阵列中的阵元数目（麦克风数）
  • MMM : 信号源数目
  • A\mathbf{A}A : 阵列流形矩阵（导向矢量组成）
  • Rx\mathbf{R}_xRx​ : 信号的自相关矩阵
  • R\mathbf{R}R : 接收信号的协方差矩阵
  • σ2\sigma^2σ2 : 噪声方差
  • B\mathbf{B}B : 酉矩阵，用于对 R\mathbf{R}R 进行对角化
  • BS\mathbf{B}_SBS​ : 与信号子空间相对应的矩阵
  • BN\mathbf{B}_NBN​ : 与噪声子空间相对应的矩阵
  • Λ1\Lambda_1Λ1​ : 信号子空间的对角矩阵
  • Λ2\Lambda_2Λ2​ : 噪声子空间的对角矩阵

推导

信号模型

Y=AX+N\mathbf{Y} = \mathbf{A} \mathbf{X} + \mathbf{N}Y=AX+N

协方差矩阵

R=E[YYH]=ARxAH+σ2I\mathbf{R} = \mathbf{E}[\mathbf{Y} \mathbf{Y}^H] = \mathbf{A} \mathbf{R}_x \mathbf{A}^H + \sigma^2 \mathbf{I}R=E[YYH]=ARx​AH+σ2I

阵列流形矩阵和信号的自相关矩阵的属性

由于 A\mathbf{A}A 是列满秩的（因为所有信号来自不同的方向）和 Rx\mathbf{R}_xRx​ 是一个非奇异的 Hermitian 矩阵， ARxAH\mathbf{A} \mathbf{R}_x \mathbf{A}^HARx​AH 也将是一个非负正定的 Hermitian 矩阵。

列满秩：所有的导向矢量是线性无关的，信号来自于不同的方向

非奇异：矩阵是可逆的，意味着矩阵的行或列是线性无关的

Hermitian: 意味着矩阵等价与其复共轭转置， Rx=RxH\mathbf{R}_x = \mathbf{R}_x^HRx​=RxH​

非负正定矩阵（Non-negative Definite Matrix）: 对于一个非负正定的 Hermitian 矩阵 M，对于任意非零向量 x，都有 xHMx≥0\textbf{x}^H\textbf{Mx}≥0xHMx≥0

证明 ARxAH\mathbf{A} \mathbf{R}_x \mathbf{A}^HARx​AH 是非负正定的 Hermitian 矩阵

Hermitian 性质:

首先我们证明 ARxAH\mathbf{A} \mathbf{R}_x \mathbf{A}^HARx​AH 是 Hermitian 的。

这里用到了 Rx\mathbf{R}_xRx​ 是 Hermitian 的这一事实，即 RxH=Rx\mathbf{R}_x^H = \mathbf{R}_xRxH​=Rx​。

非负正定性:

要证明 ARxAH\mathbf{A} \mathbf{R}_x \mathbf{A}^HARx​AH 是非负正定的，我们需要证明对于任意非零向量 x\mathbf{x}x，都有 xH(ARxAH)x≥0\mathbf{x}^H (\mathbf{A} \mathbf{R}_x \mathbf{A}^H) \mathbf{x} \geq 0xH(ARx​AH)x≥0。

令 y=AHx\mathbf{y} = \mathbf{A}^H \mathbf{x}y=AHx, 则

因为 Rx\mathbf{R}_xRx​ 是非负正定的，所以 yHRxy≥0\mathbf{y}^H \mathbf{R}_x \mathbf{y} \geq 0yHRx​y≥0。

综上, ARxAH\mathbf{A} \mathbf{R}_x \mathbf{A}^HARx​AH 是一个非负正定的 Hermitian 矩阵,大小为P×PP \times PP×P， 且秩为MMM。

那么根据谱定理，此时R\textbf{R}R 一定可以被一个酉矩阵B\textbf{B}B对角化， 即BHRB\textbf{B}^H \textbf{R} \textbf{B}BHRB 是对角的，那么有BBH=I\textbf{B} \textbf{B}^H = \textbf{I}BBH=I ， B\textbf{B}B 是一个P×PP \times PP×P的酉矩阵，且B=[Bs,BN]\textbf{B} = [\textbf{B}_s, \textbf{B}_N]B=[Bs​,BN​]

BHRB=BH(ARxAH+σ2I)B=Diag[(μ1+σ2),...,(μM+σ2),σ2,...,σ2]=Diag[Λ1,Λ2]\textbf{B}^H\textbf{R} \textbf{B} = \textbf{B}^H(\mathbf{A} \mathbf{R}_x \mathbf{A}^H+\sigma^2\textbf{I}) \textbf{B} = Diag[(\mu_1 + \sigma^2),...,(\mu_M + \sigma^2),\sigma^2,...,\sigma^2] = Diag[\Lambda_1, \Lambda_2]BHRB=BH(ARx​AH+σ2I)B=Diag[(μ1​+σ2),...,(μM​+σ2),σ2,...,σ2]=Diag[Λ1​,Λ2​], 这里Λ1\Lambda_1Λ1​包含前MMM 个最大的特征值，与信号相关，这里Λ2\Lambda_2Λ2​ 包含了剩余的P−MP-MP−M 个特征值，与噪声相关。

那么将上式重写可以得到：

利用 BHRB=Diag[Λ1,Λ2]\mathbf{B}^H \mathbf{R} \mathbf{B} = \text{Diag}[\Lambda_1, \Lambda_2]BHRB=Diag[Λ1​,Λ2​]，我们有：

ARxAHBS=BSΛ1−BS(σ2I)\mathbf{A} \mathbf{R}_x \mathbf{A}^H \mathbf{B}_S = \mathbf{B}_S \Lambda_1 - \mathbf{B}_S (\sigma^2 \mathbf{I})ARx​AHBS​=BS​Λ1​−BS​(σ2I)

ARxAHBN=BNΛ2−BN(σ2I)=0\mathbf{A} \mathbf{R}_x \mathbf{A}^H \mathbf{B}_N = \mathbf{B}_N \Lambda_2 - \mathbf{B}_N (\sigma^2 \mathbf{I}) = 0ARx​AHBN​=BN​Λ2​−BN​(σ2I)=0

从上面的第二个方程，我们可以推断 AHBN=0\mathbf{A}^H \mathbf{B}_N = 0AHBN​=0 。这就是说，阵列流形矩阵 A\mathbf{A}A （或它的列，即导向矢量）和噪声子空间 BN\mathbf{B}_NBN​ 是正交的。

结论

这样，我们就证明了导向矢量（或阵列流形矩阵 A\mathbf{A}A 的列）和噪声子空间是正交的。这个性质是许多高分辨率方向估计算法（如 MUSIC、ESPRIT 等）的基础
