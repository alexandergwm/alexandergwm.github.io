解惑：为什么子空间算法中的导向矢量张成的信号子空间与噪声子空间正交

发表于2023-08-26|更新于2024-05-05|知识总结
|阅读量:

为什么子空间算法中的导向矢量张成的信号子空间与噪声子空间正交

MUSIC算法为代表的子空间算法算是信号处理和阵列信号处理下的一个常用的算法，其关键思想在于将接收到的信号分为两个正交子空间：信号子空间和噪声子空间。信号子空间包含的是从各个方向到达阵列的信号的信息，而噪声子空间主要包含噪声和干扰，当进行特征值分解（EVD）或者奇异值分解（SVD）时，这两个子空间由对应的特征向量定义。

那么为什么子空间算法中的导向矢量张成的信号子空间和噪声子空间正交？

我们从数学公式上来进行推导，先对每个符号的含义做一个定义。

当然，让我们将你的详细推导与我之前的建议结合起来，以构建一个全面而详细的解释和证明。

符号与定义

  • $P$ : 阵列中的阵元数目（麦克风数）
  • $M$ : 信号源数目
  • $\mathbf{A}$ : 阵列流形矩阵（导向矢量组成）
  • $\mathbf{R}_x$ : 信号的自相关矩阵
  • $\mathbf{R}$ : 接收信号的协方差矩阵
  • $\sigma^2$ : 噪声方差
  • $\mathbf{B}$ : 酉矩阵，用于对 $\mathbf{R}$ 进行对角化
  • $\mathbf{B}_S$ : 与信号子空间相对应的矩阵
  • $\mathbf{B}_N$ : 与噪声子空间相对应的矩阵
  • $\Lambda_1$ : 信号子空间的对角矩阵
  • $\Lambda_2$ : 噪声子空间的对角矩阵

推导

信号模型

$\mathbf{Y} = \mathbf{A} \mathbf{X} + \mathbf{N}$

协方差矩阵

$\mathbf{R} = \mathbf{E}[\mathbf{Y} \mathbf{Y}^H] = \mathbf{A} \mathbf{R}_x \mathbf{A}^H + \sigma^2 \mathbf{I}$

阵列流形矩阵和信号的自相关矩阵的属性

由于 $\mathbf{A}$ 是列满秩的（因为所有信号来自不同的方向）和 $\mathbf{R}_x$ 是一个非奇异的 Hermitian 矩阵， $\mathbf{A} \mathbf{R}_x \mathbf{A}^H$ 也将是一个非负正定的 Hermitian 矩阵。

列满秩：所有的导向矢量是线性无关的，信号来自于不同的方向

非奇异：矩阵是可逆的，意味着矩阵的行或列是线性无关的

Hermitian: 意味着矩阵等价与其复共轭转置， $\mathbf{R}_x = \mathbf{R}_x^H$

非负正定矩阵（Non-negative Definite Matrix）: 对于一个非负正定的 Hermitian 矩阵 M，对于任意非零向量 x，都有 $\textbf{x}^H\textbf{Mx}≥0$

证明 $\mathbf{A} \mathbf{R}_x \mathbf{A}^H$ 是非负正定的 Hermitian 矩阵

Hermitian 性质:

首先我们证明 $\mathbf{A} \mathbf{R}_x \mathbf{A}^H$ 是 Hermitian 的。

这里用到了 $\mathbf{R}_x$ 是 Hermitian 的这一事实，即 $\mathbf{R}_x^H = \mathbf{R}_x$。

非负正定性:

要证明 $\mathbf{A} \mathbf{R}_x \mathbf{A}^H$ 是非负正定的，我们需要证明对于任意非零向量 $\mathbf{x}$，都有 $\mathbf{x}^H (\mathbf{A} \mathbf{R}_x \mathbf{A}^H) \mathbf{x} \geq 0$。

令 $\mathbf{y} = \mathbf{A}^H \mathbf{x}$, 则

因为 $\mathbf{R}_x$ 是非负正定的，所以 $\mathbf{y}^H \mathbf{R}_x \mathbf{y} \geq 0$。

综上, $\mathbf{A} \mathbf{R}_x \mathbf{A}^H$ 是一个非负正定的 Hermitian 矩阵,大小为$P \times P$， 且秩为$M$。

那么根据谱定理，此时$\textbf{R}$ 一定可以被一个酉矩阵$\textbf{B}$对角化， 即$\textbf{B}^H \textbf{R} \textbf{B}$ 是对角的，那么有$\textbf{B} \textbf{B}^H = \textbf{I}$ ， $\textbf{B}$ 是一个$P \times P$的酉矩阵，且$\textbf{B} = [\textbf{B}_s, \textbf{B}_N]$

$\textbf{B}^H\textbf{R} \textbf{B} = \textbf{B}^H(\mathbf{A} \mathbf{R}_x \mathbf{A}^H+\sigma^2\textbf{I}) \textbf{B} = Diag[(\mu_1 + \sigma^2),…,(\mu_M + \sigma^2),\sigma^2,…,\sigma^2] = Diag[\Lambda_1, \Lambda_2]$, 这里$\Lambda_1$包含前$M$ 个最大的特征值，与信号相关，这里$\Lambda_2$ 包含了剩余的$P-M$ 个特征值，与噪声相关。

那么将上式重写可以得到：

利用 $\mathbf{B}^H \mathbf{R} \mathbf{B} = \text{Diag}[\Lambda_1, \Lambda_2]$，我们有：

$\mathbf{A} \mathbf{R}_x \mathbf{A}^H \mathbf{B}_S = \mathbf{B}_S \Lambda_1 - \mathbf{B}_S (\sigma^2 \mathbf{I})$

$\mathbf{A} \mathbf{R}_x \mathbf{A}^H \mathbf{B}_N = \mathbf{B}_N \Lambda_2 - \mathbf{B}_N (\sigma^2 \mathbf{I}) = 0$

从上面的第二个方程，我们可以推断 $\mathbf{A}^H \mathbf{B}_N = 0$ 。这就是说，阵列流形矩阵 $\mathbf{A}$ （或它的列，即导向矢量）和噪声子空间 $\mathbf{B}_N$ 是正交的。

结论

这样，我们就证明了导向矢量（或阵列流形矩阵 $\mathbf{A}$ 的列）和噪声子空间是正交的。这个性质是许多高分辨率方向估计算法（如 MUSIC、ESPRIT 等）的基础
