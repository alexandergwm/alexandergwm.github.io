摘要

​ 已经以多种方式解决了使用一个或多个源和麦克风阵列根据室内测量来估计声反射系数的问题。一种特别说明性的方法是平面波分解。该表示中的峰值可以分配给镜像源的位置和强度。传统方法采用刚性球形麦克风阵列和模态波束形成。压缩感知（CS）技术通过假设某些给定表示中声场的稀疏性来实现空间欠采样。通过仔细设计传感矩阵，与奈奎斯特采样和重建相比，所需测量的数量可以大大减少。然而，通常没有给出当前问题的稀疏性，并且违反了传感矩阵的足够低的相互相干性，从而产生鲁棒性较低的CS重建。我们的目标是通过使用刚性球形麦克风阵列数据的 CS 来稳健地检测早期离散反射。我们讨论了房间特征、传感矩阵和技术措施的相互作用，以限定检测性能。研究了实际限制的影响，例如麦克风的数量及其自噪声，以及与传统模态波束形成技术相比，将 CS 应用于该问题的总体增益

关键词：早期房间反射、麦克风阵列、声场分析、模态波束形成、压缩感知

介绍

使用传感器阵列进行到达方向（DOA）估计是阵列处理中一个很好解决的问题，对此存在各有利弊的不同方法[1]。球形麦克风阵列允许使用球谐函数 (SH) 在模态域中通过波束成形来进行 DOA 估计。随着高计算能力的出现，具有特定权重（例如匹配滤波器特性、延迟求和、最小方差无失真响应）的模态平面波分解（PWD）成为刚性和开放球麦克风阵列测量的有用工具，具有以下特点：全音频带宽[2]。在过去十年中，压缩传感 (CS) 技术 [3] 越来越多地用于解决声学问题 [4]。 CS 允许对信号进行高度欠采样，并在理想情况下实现完美重建，但前提是在某个（未知和/或不可访问）信号域中保持稀疏性。自然地，CS 很快就应用于声学波束形成和 DOA 估计，例如[5, 6]，在所提出的场景中优于上述方法。例如，通过 CS 重建混响声场。在[7, 8]中，表明对于声场的早期部分和非常低的音频有非常好的重建。

​ 在这篇文章中，我们的目标是对早期房间反射 [9] 进行可靠的 DOA 估计，即房间脉冲响应的峰值比房间的所谓混合时间 [10] 早得多到达。我们将传统的 PWD 方法与基于 CS 的方法进行比较。由于半径范围为 415 厘米的刚性球形麦克风阵列如今可用于实际的现场房间测量，因此我们专注于这些设计，而不是随机采样或更大孔径的麦克风阵列。问题是，在这种情况下，将 CS 而非 PWD 方法用于 DOA 估计是否可以盈利？🔤

方法和设置

我们通过一个具体的例子来进行评估，主要遵循研究[9]，其中房间反射的 DOA 及其各自的墙壁吸收系数由 PWD 估计。所讨论的房间的尺寸为（宽度x，长度y，高度z）=（10,11,3）米。源位于 $x_S = (7, 8, 1.8)^Tm$，其中 $x = (x, y, z)^T$ 表示笛卡尔坐标中的向量。接收器位置为 $x_R = (2.1, 4.5, 1.8)^Tm$。房间墙壁应由砖砌成，即假设频率无关吸收系数为 0.15。模拟使用 fs = 16 kHz 的采样频率和 c = 343 m/s 的声速。感知混合时间被预测为[10] $t_{mp,50 %} = 31.2ms$ 和 $t_{mp,95%} = 54ms$ 。因此，我们将讨论限制在时间窗口 $0 ≤ t ≤ 40 ms$，其中“入射角随时间变化”的反射稀疏性假设应该是有效的。对于这个时间窗口，直达声和来自多达 4 阶镜像源的 31 个反射到达麦克风阵列，参见。图 5 中的详细列表。仅对于图 1 中讨论的简化和介绍性场景，使用具有完全吸收地板和天花板（而不是砖块）的一阶（而不是四阶）图像源模型，同时保持所描述的所有以上其他参数。在声脉冲响应中，这仅导致水平面上的一阶反射。

​ 平面波入射和麦克风阵列接收器之间的时域声学传输矩阵 $A_{N×t×M}$（在 CS 中通常称为传感矩阵）是由平面波入射的固定球形 Lebedev 网格的 $M = 1202 $个角度对（方位角$ \Phi_m$，余度 $θ_m$）构建的N = 14、26、38 或 170 个麦克风，排列在球形 Lebedev 网格 ($φ_n$,$ θ_n$) 上，半径 $R_N = 0.1 m$，中心位于位置 $x_R$。如上所述，我们不对麦克风阵列位置部署随机采样，因为我们对使用已经构建的刚性球体麦克风阵列相当感兴趣。因此，对接收器和 DOA 字典都使用 Lebedev 网格会导致相当高的相互一致性。请注意，刚性球形麦克风阵列上平面波入射的蒙特卡洛模拟（此处未讨论）表明，与常规采样方案相比，随机采样不一定会降低相互相干性。

​ 来自 $(φ_{is}, θ_{is})$ 撞击到刚性麦克风阵列 $(φ_n, θ_n)$ 的房间反射的测量矩阵 $y_{N×t}$ 通过图像源模型获得，并使用足够的时间分辨率在 SH 和时间频域中实现。逆空间和时间傅立叶变换产生$ N$ 个麦克风的声学脉冲响应$ y_{N×t}$。然后添加具有专用信号峰噪声方差比的不相关高斯白噪声（如果有）来模拟传感器噪声。请注意，这个简单的加性噪声模型仅适用于脉冲激励的脉冲响应测量，参见。 [11]。

​ 通过平面波分解进行的 DOA 估计包括 SH 域中的匹配滤波，产生高频独立的白噪声增益，参见[9] 详细信息。通过对所有平面波字典进行 SH 逆变换获得 N = 14、26、38、170 个麦克风网格的 PWD 矩阵 $p[N]_{φ_m×θ_m×t}$ 及其保守选择的相应 SH 阶数 2、3、4、10条目（$φ_m，θ_m$）。通常，随着时间的推移，任一角度和时间的对应都可以在曲面图中方便地可视化，参见。 [9，图 6]，或$ θ $与 $φ $的关系是针对固定时间实例绘制的，参见。 [2，图5.6]。

​ 通过压缩感知的 DOA 估计使用正交匹配追踪 (OMP) [3，第 4章]在Python的scikit-learn包中实现。对于所有考虑的 $t$，长度$ t_w = 0.75 ms$（根据选择的声程 2.5725$R_N$，即 $f_s$ 处的 12 个样本）的滑动窗口用于求解方程组 $y_{N·tw×1} = A_{N·tw×M}· x_{M×1}$（矩阵/向量堆叠），用于使用 OMP 的 $x_{M×1}$。我们在每个时间帧中搜索 $x_{M×1}$ 的 3 个非零系数。这似乎是任意的，但却与某些反射的时间巧合有关。因此，这种选择很大程度上取决于房间的特性。然后，针对特定的$t$找到特定的OMP解，得到具有3$M$个非零系数的稀疏矩阵$x_{M×t}$。明确地重新排列平面波入射角并考虑不同的麦克风阵列网格 N，使用 CS-OMP 波束成形给出 DOA 矩阵 $x[N]_{φm×θm×t}$。

​ 请随意使用本文的专用 Jupyter Notebook1 进行自己的模拟。

结果和讨论

下面我们的目标是使用相同的设置直接比较 PWD 和基于 CS 的 DOA 矩阵 $p[N]{φm×θm×t}$ 和 $x[N]{φm×θm×t}$。这些原始数据可用于计算 DOA 估计，例如斑点或峰值检测。

​ 对于随时间变化的三维 (3D) DOA 结果，早期反射的四个维度 $φ$ 、$ θ$ 、 $t $和$level$将被可视化以进行呈现。以下散点图似乎是一种合理的方法。请注意，法线曲面图很容易提供 4D 数据的误导性图片，尤其是在稀疏时。图中左边的散点图将 DOA 显示为水平与方位角随时间的变化。右侧散点图将 DOA 显示为随时间变化的余纬度水平。顶部散点图显示基于 PWD 的 DOA 矩阵 $|p_{φ_m×θ_m×t }|$以分贝为单位。底部散点图显示基于 CS-OMP 的 DOA 矩阵 $|x_{φ_m×θ_m×t} |$以分贝为单位。直达声的level标准化为 0 dB，可见范围为 -10 dB 至 1 dB。假设镜像源/反射的行为类似于点源，则可以补偿它们随距离 r 的幅度衰减 ∝ 1/r。这可以简单地通过从直达声到达开始的行进时间 $t$ 估计 $\frac{1}{r} = \frac{1}{ct}$ 来实现。该方法实际上仅适用于一阶图像源。否则，对于高阶图像源（多次壁反射），吸收系数必须相当低，以便这种补偿有意义。

​ 散点圆圈的颜色和大小表示水平。每个子图的左上角包含两个参考圆，不要与估计的反射混淆：大黄色圆圈表示 0 dB 的大小,电平，小黑圈-10 dB。圆之间的半径以 dB 为单位线性变化。一个小白点和其正上方的绿色数字表示图像源的参考位置。编号是根据travelling时间排序的。将来可以通过将低水平叠加到高水平来改进散点图。

​ 由于参数变化的自由度很大，我们将讨论限制在具有不同特征的四种典型场景。第一种情况的结果如图 1 所示，其中 N = 170 和一阶图像源位于具有完全吸收天花板和地板的房间（参见有白雪覆盖的草地的庭院）。这个相当人为的场景链接到[9]中使用的设置，旨在引入可视化策略。可以看出，PWD 和 CS-OMP 都可以可靠地检测到所有五个 DOA（由于 $|x_S − x_R|$ = 6.02 m 以及来自四个墙壁的四个反射，在 17.556 ms 处直达声）水平面。在此可视化中，PWD 在电平最大值（此处即主瓣）周围呈现典型的椭圆形状，对应于映射在球体上的圆形波束图案。在搜索与椭圆/圆形焦点匹配的最大电平时，稳健的斑点检测将依赖于这些形状的清晰分离（没有光栅和旁瓣的拖尾）。 OMP 的设计展示了稀疏的 DOA 事件。即使在这种简单的场景中，也会发生某些 DOA 的多次检测，参见。围绕事件 1 和 2 的突出稍微错位，但这不会提供误导性结果，因为它们可以清楚地区分大约同一时间实例的最大级别条目。

​ 图 2 的第二个场景使用 N = 170 个接收器，所有墙壁均由砖块和高达 4 阶的图像源构建。因此，需要估计直达声和 31 个反射的 DOA。通过目视检查直至第 11 个镜像源，可以清楚地区分 PWD 内的椭圆。 33 ms 左右的簇和大约 39 毫秒的簇可能无法检测到，而在某些反射之间明显是孤立的，参见。镜像源 20。这又高度依赖于房间特性。如果正确找到 DOA，基于 CS-OMP 的 DOA 就会非常精确，这里的情况总是如此。

​ 图 3 的第三个场景使用 N = 170 个接收器，所有墙壁均由砖砌成，图像源高达 4 阶，并且添加到接收器的不相关白噪声具有 10 dB 的相当高的峰噪比。选择此选项是为了在比较我们的 PWD 和 CSOMP DOA 方法时方便地讨论噪声测量的一般影响。回想一下，我们补偿了$ \frac{1}{r}$ 振幅衰减，它会随着时间的增加而增加噪声。这会降低 PWD 以及 CS-OMP 中的 DOA 结果。但特点却不同。目视检查允许对 PWD 中的第 11 个事件进行稳健的 DOA 估计，与没有噪声的示例相同。这些椭圆不会因噪声而退化太多。基于 CS-OMP 的 DOA 在 t > 27.5 ms 内变得模糊。此外，前 5 个镜像源并非无可争议，但对于稳健的 DOA 估计来说并不重要。然而，这种模糊会导致 DOA 聚类的数量随着时间的增加而增加。虽然 PWD 估计将确保对前 11 个镜像源的稳健估计，但这对于本例中的 CS-OMP 来说并不适用，参见。事件 11、17 和 18 周围的误报率较高。相反，使用 CS-OMP 可以对例如事件进行稳健的检测。在非常嘈杂的数据中可能会发生事件 14，其中 PWD 数据将失败。

图 4 的第四个场景使用 N = 14 个接收器，所有墙壁均由砖块和高达 4 阶的图像源构建，无噪声。因此，CS 的先决条件$ N<< M$ 得到了最好的满足。此外，N 约为DOA 估计的早期反射数量的一半，即使没有噪声，也应被视为具有挑战性的场景。通过目视检查基于 PWD 数据的稳健 DOA 估计对于直达声和前两次反射是有意义的。第四和第五事件不能可靠地分开。基于 CS-OMP 的 DOA 是模糊的。模糊特性与前面的示例不同。对于 N = 14，由于稀疏空间采样，模糊分布在整个时间范围内；而对于 N = 170，由于 $1/r$ 补偿，模糊随着时间的增加而变得更加突出，从而增加了噪声水平。对于事件 1 至 9，可以使用 CS-OMP 进行鲁棒检测。对于所有其他事件，在盲目分配早期反射时，误报的水平不容忽视。例如，请注意，24 ms 处的误报水平相当高，表明除了第 4 和第 5 镜像源之外还有进一步的反射。
